The Original EggBot: Software Repository

Intro

This is a repository for software to drive The Original EggBot kit, by Evil Mad Scientist Laboratories. More information at http://www.egg-bot.com/

This software hosted here includes the driver extensions for Inkscape, additional helper extensions, firmware for the EiBotBoard (AKA "EBB", or EggBotBoard), and example plot files. All code is designed to work on Mac, Windows and Linux, and with all existing models of EggBot.

Download and install

To download and install this software, please see the detailed instructions on our main documentation wiki.
